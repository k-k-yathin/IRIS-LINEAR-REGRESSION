## Aim
To implement Linear Regression using the IRIS dataset and understand supervised learning and regression analysis.

## Dataset
IRIS Dataset containing sepal length, sepal width, petal length, and petal width.

## Tools Used

- Pandas
- NumPy
- scikit-learn
- Matplotlib

## Steps
1. Load IRIS dataset from CSV
2. Select numerical features and target variable
3. Split data into training and testing sets
4. Train Linear Regression model

The model successfully predicts the target variable and performance is evaluated using standard regression metrics.
